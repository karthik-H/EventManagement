### 2. Task Management
- Create a task with:
  - Task title
  - Description
  - Status (To Do, In Progress, Completed)
- View all tasks
- Edit task details
- Delete a task

---

### 3. Task Assignment to Events
- Assign one or more tasks to an event
- View all tasks associated with a specific event
- Reassign a task to another event
- Remove a task from an event
- Each task must be associated with exactly one event
